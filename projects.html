CodingCore is a website I made for myself. I use a lot of websites when coding and I'm too lazy to keep typing in all my searches, so I just made this website to be able to press buttons instead of having to type everything out.

Development started July 8, 2022 and is still ongoing. I've separated version 0.8 and 2.0, as I wanted to rewrite codingcore, but keep the original version.
